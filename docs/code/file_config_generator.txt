FIchier de configuration du générateur
______________

Ce fichier explique la construction et les différentes options pour le fichier de configuration du générateur.

Ils se retrouve de la forme :
<params>
    <shape name=... option1=... option2=... />
    <editor name=... option1=... option2=... />
</params>

On a du coup les différentes options de la forme basique à la suite des uns des autres ainsi que pour l'éditeur.

Un exemple minimaliste est le suivant :
<?xml version="1.0" encoding="UTF-8"?>
<params>
    <shape name="icosphere" />
    <editor name="noisy_height" />
</params>
